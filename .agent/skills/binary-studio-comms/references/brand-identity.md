# Brand Identity — Binary Studio

## Logo

### Logo Completo
**B1NARY STUDI0** — El "1" en la B y el "0" al final son en verde (#00CC66) sobre fondo oscuro. El resto del texto es blanco. La fuente del logo es Space Grotesk.

Archivo: `assets/logo-full.svg`

### Logo Reducido
**01** — Versión mínima del logo. El "0" tiene un trazo diagonal interno. Ambos caracteres en verde (#00CC66) sobre fondo negro.

Archivo: `assets/logo-reduced.svg`

### Reglas de Uso del Logo
- Fondo mínimo: usar siempre sobre fondos oscuros (#202020 o más oscuro)
- No alterar las proporciones ni los colores del logo
- Espacio de respeto: mínimo un "0" de distancia a los bordes
- No colocar el logo sobre fondos que compitan con el verde
- Versiones permitidas: completo o reducido (no inventar variantes)

## Paleta de Colores

### Colores Principales

| Nombre | Hex | Uso |
|--------|-----|-----|
| **Negro Binary** | `#202020` | Fondo principal, base de toda la marca |
| **Verde Binary** | `#00CC66` | Color primario, acentos, CTAs, elementos interactivos |
| **Magenta Binary** | `#CC0A82` | Color secundario, gradientes, highlights, variedad visual |
| **Amarillo Binary** | `#F8F800` | Color terciario, alertas, acentos energéticos, gradientes |
| **Blanco** | `#FFFFFF` | Texto sobre fondos oscuros |

### Uso de Colores

- **Fondo:** Siempre #202020 como base. Nunca usar fondos blancos o claros como identidad principal
- **Verde (#00CC66):** Color dominante después del negro. Usar en títulos, botones, iconos, elementos destacados
- **Magenta (#CC0A82):** Para generar contraste y variedad. Funciona bien en gradientes con el verde o el amarillo
- **Amarillo (#F8F800):** Uso moderado. Ideal para gradientes, highlights, y como energía visual
- **Gradientes permitidos:** Verde → Amarillo, Magenta → Amarillo, Verde → Magenta

### Combinaciones de Fondo

1. **Negro sólido** (#202020) — Uso principal
2. **Gradiente verde-amarillo** — Para posts energéticos (tips, recursos)
3. **Gradiente magenta-oscuro** — Para posts de diseño, portfolio
4. **Negro con elementos de color** — Acentos de color sobre fondo oscuro

## Tipografía

### Thunder
- **Uso:** Títulos, headlines, display text
- **Características:** Bold, condensada, impactante
- **Contexto:** Posts de redes sociales, títulos de secciones web, titulares
- **Siempre en mayúsculas** cuando se usa como headline principal

### Space Grotesk
- **Uso:** Cuerpo de texto, subtítulos, UI, párrafos
- **Características:** Geométrica, moderna, legible
- **Contexto:** Captions, descripciones, contenido web, interfaz
- **Pesos sugeridos:** Regular (400) para cuerpo, Medium (500) para subtítulos, Bold (700) para énfasis

### Reglas Tipográficas
- No usar otras fuentes salvo para contenido citado o de terceros
- Títulos principales: Thunder Bold, uppercase
- Subtítulos: Space Grotesk Medium
- Cuerpo: Space Grotesk Regular
- Jerarquía clara: máximo 3 niveles de tamaño por pieza

## Iconografía

- **Librería principal:** Radix Icons
- **Alternativas:** Lucide, Phosphor Icons (mantener estilo consistente de línea fina)
- **Estilo:** Outline/línea, no filled/sólido
- **Color:** Verde Binary o Blanco según contexto
- **Tamaño consistente** dentro de cada pieza

---

## Estilo Visual

### Fotografía
- Preferir fotos en blanco y negro o con tratamiento de alto contraste
- Cuando se usa foto a color, aplicar overlay de los colores de marca
- Estilo editorial, no stock genérico

### Composición
- Uso de grillas y sistemas modulares
- Contrastes fuertes entre tipografía y fondo
- Espacio negativo como recurso de diseño
- Elementos geométricos (cuadrados, rectángulos) alineados con la estética "binary/digital"

### Elementos Gráficos
- Cuadrados/rectángulos pequeños en verde como elementos decorativos (visibles en la grilla actual de contenido)
- Barras de categoría en la parte superior de los posts (ej: "Tips y recursos →", "Código →")
- Etiquetas de categoría como badges pequeños (ej: "+ DISEÑO", "+ CÓDIGO")
